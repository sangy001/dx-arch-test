# arch-test

- config: 测试时需要添加的基本参数
- data: 接口测试的数据
- element: ui测试时，各页面的元素定位
- image-build: 构建测试环境镜像的dockerfile
- install-env: 安装Arch测试环境的执行脚本以及相关配置
- testcase: 实际执行的测试用例
- main.py: 执行函数
- pytest.ini: pytest的配置文件
- send_message.py: 发送企业微信robot的信息
